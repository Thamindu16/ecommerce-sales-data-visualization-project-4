# E-Commerce Sales Data Visualization and Insights Using Python

## Project Overview

    This project was completed as part of my Data Analytics Internship at DecodeLabs.
    
    The objective of this project was to explore and visualize an E-Commerce sales dataset using Python. Data visualization techniques were used to better understand customer behavior, sales performance, product popularity, payment preferences, and revenue trends.
    
    Before creating visualizations, the dataset was inspected and prepared to ensure data quality. Various charts and graphs were generated using Matplotlib and Seaborn to transform raw data into meaningful insights.
    
    The project demonstrates how data visualization can help identify patterns, trends, and business opportunities from sales data.

## Technologies Used

    * Python
    * Pandas
    * Matplotlib
    * Seaborn
    * Google Colab
    * Microsoft Excel

## Data Preparation

    The following steps were completed before creating visualizations:
    
      * Dataset inspection
      * Understanding dataset structure
      * Missing value detection
      * Handling missing values
      * Data validation and preparation

## Visualizations Created

    The following visualizations were generated during the analysis:
    
      * Product Sales Analysis
      * Payment Method Distribution
      * Order Status Distribution
      * Referral Source Analysis
      * Monthly Sales Trend Analysis
      * Revenue by Product
      * Total Price Distribution

## Key Insights

    The visualizations helped identify several important patterns within the dataset:
    
      * Certain products showed higher sales volumes than others.
      * Customer payment preferences were clearly visible through payment method analysis.
      * Monthly sales trends highlighted fluctuations in business performance.
      * Revenue analysis revealed the products contributing most to total sales.
      * Referral source analysis provided insights into customer acquisition channels.
      * Order status distribution helped evaluate overall transaction outcomes.

## How to Run the Project

    1. Open the notebook in Google Colab.
    2. Upload the dataset file.
    3. Run all notebook cells sequentially.
    4. Generate the visualizations.
    5. Review the charts and analysis results.
